정보 확인 가이드

노트북 정보 확인 방법

노트북 바닥 면 스티커 확인 (4단계)

대부분의 정보는 노트북 바닥 면의 스티커에 모두 표기되어 있습니다.

💻
1. 바닥 면 확인

노트북을 뒤집어 바닥 면을 확인합니다.

🔍
2. 스티커 확인

바닥에 붙어있는 모델명/규격 스티커를 찾습니다.

🏷️
3. S/N 확인

'S/N' 또는 '일련번호' 항목의 값을 확인합니다.

📅
4. 제조연월 확인

'제조연월' 또는 'MFD' (YYYY.MM) 값을 확인합니다.

명령 프롬프트(cmd)로 시리얼 넘버 확인

스티커가 훼손되었거나 글씨를 읽기 어려운 경우, 아래 방법으로 시리얼 넘버(S/N)를 확인할 수 있습니다. (제조연월은 확인되지 않을 수 있습니다.)

1. 키보드의 Windows 키 + R 키를 눌러 '실행' 창을 열거나, 시작 메뉴 검색창에서 cmd를 검색하여 '명령 프롬프트'를 실행합니다.

2. 검은색 명령 프롬프트 창이 나타나면, 아래 회색 박스 안의 명령어를 복사하여 붙여넣기 한 후 Enter 키를 누릅니다.

wmic bios get serialnumber

3. 'SerialNumber' 라는 텍스트 아래에 영문/숫자로 조합된 값이 나타납니다. 이 값이 노트북의 시리얼 넘버입니다.

Apple 메뉴()로 시리얼 넘버 확인

macOS 사용자는 이 방법이 가장 간단합니다. (바닥 면 각인으로도 확인 가능)

  1. 화면 좌측 상단의 Apple 로고()를 클릭합니다.
  2. 드롭다운 메뉴에서 '이 Mac에 관하여(About This Mac)'를 선택합니다.
  3. 나타나는 창의 '개요(Overview)' 탭에 '일련번호(Serial Number)'가 표시됩니다.
  4. 제조연월 확인:
    • 이 메뉴에서는 제조연월이 확인되지 않습니다.
    • 노트북 바닥 면의 각인을 확인하시거나, 확인이 어려운 경우 '비고'란에 '확인 불가'로 기재해 주세요.

모니터 정보 확인 방법

모니터 뒷면 스티커 확인 (4단계)

모니터의 정보는 대부분 제품 뒷면의 스티커에 표기되어 있습니다. (전원 케이블 연결부 근처)

🔍

뒷면에 붙어있는 모델명/규격 스티커를 찾습니다.

🏷️
3. S/N 확인

'S/N' 또는 '일련번호' 항목의 값을 확인합니다.

📅
4. 제조연월 확인

'제조연월' 또는 'MFD' (YYYY.MM) 값을 확인합니다.
